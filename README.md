## 🏗️ Got some ideas in docs/design

## Setup

* Must install
  * `protoc`
  * `cargo`/`rust`

* Run server
  
  ```shell
  cargo run -- -b 0.0.0.0:7331
  ```

* Run client

  ```shell
  cargo run --bin client -- -a http://127.0.0.1:7331
  ```
